1) Get a github user account. Get one here: https://github.com/
2) Let the owner of the repository know you want write access to the datacall repository
3) Install git (https://git-scm.com/) and e.g. tortoiseGit for easy committing to the git repository (https://tortoisegit.org/)
4) Create a local folder on your machine, e.g. on your C or D drive
5) Clone the specific github site to this folder. The url will be displayed once you click the 'clone or download' green button on this page (https://github.com/nielshintzen/datacallsVMSLogbook)
6) Automatic initialisation should start
7) If you want to add / delete code use commit + push
8) If you want to onl download, use the pull command
